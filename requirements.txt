-------------------------------------------------------------------
Finalized Algorithm Recipe
Categorical Matching (Base Points)

+2.0 points if song's genre matches favorite_genre
+1.0 point if song's mood matches favorite_mood
Numeric Feature Similarity (Scaled Points)

For each numeric feature, compute similarity as 1 - abs(song_value - target_value) / max_range
energy: max_range = 1.0 (0-1 scale)
tempo_bpm: max_range = 200 (0-200 bpm)
valence: max_range = 1.0
danceability: max_range = 1.0
acousticness: max_range = 1.0
Multiply each similarity by a weight (e.g., 1.0 for all) and sum them
Total similarity points: sum of weighted similarities (range 0-5.0 if 5 features × 1.0 weight)
Total Score

Total = genre_points + mood_points + similarity_points
Range: 0 to 8.0 (2 + 1 + 5)
Ranking and Selection

Sort songs by total score descending
Recommend top N songs (e.g., top 5)
If ties, break by closest tempo match, then energy match
Filtering (Optional)

Only recommend songs with total score ≥ 3.0 to ensure relevance

-------------------------------------------------------------------

## Data Flow Map

**Input (User Prefs)**  
→ Load taste profile: `favorite_genre`, `favorite_mood`, target values for `energy`, `tempo_bpm`, `valence`, `danceability`, `acousticness`  
→ Read CSV file: `songs.csv` with song data  

**Process (The Loop: Judging Every Song)**  
→ For each song in CSV:  
  - Check genre match: +2.0 if matches `favorite_genre`  
  - Check mood match: +1.0 if matches `favorite_mood`  
  - Compute similarities: For each numeric feature, calculate `1 - abs(song_value - target_value) / max_range`  
  - Sum similarities (weighted, e.g., ×1.0 each)  
  - Total score = genre_points + mood_points + similarity_sum  
→ Store score for each song  

**Output (The Ranking: Top K Recommendations)**  
→ Sort songs by total score descending  
→ Select top K (e.g., top 5)  
→ Return list: song titles, artists, scores, and why they match (e.g., "Genre match + high energy similarity")  

This flow processes all songs once per recommendation, ensuring efficient comparison against user preferences.
